Tải về và chạy xem:

B1: mở folder ProjectWeb ra sẽ thấy 2 folder

B2: Nhấp chuột phải vào folder badminton-store-main, sau đó chọn "Show more option" và nhấp vào "Open" để mở folder đó trên công cụ Lập trình của bạn (làm tương tự bước 2 với folder còn lại)

B3: Sau khi mở folder trên công cụ lập trình thì chọn "Terminal -> New terminal". Và gõ các lần lượt các lệnh sau:

  "npm install"
	
  "npm install express mongoose cors"
	
  "npm install --save-dev nodemon"
	
  "npm run dev"
	
(CHÚ Ý: B3 này làm lần lượt từng folder một)

LƯU Ý: Cần cài đặt MongoDB về laptop hoặc PC của bạn thì chương trình mới chạy được !
